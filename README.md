# GenZen

Avec GenZen, les enfants gagnent du temps d’écran en accomplissant des missions réelles (lire, marcher, aider à la maison…). Les parents créent et valident ces missions, et chaque mission réussie débloque du temps d’écran pour le lendemain.
Pour l’enfant :
- Une liste de missions claires et motivantes.
- Un compteur de temps d’écran à gagner.
- Une validation simple par les parents.

Pour les parents :
- Un outil pour encourager les activités hors écran.
- Un contrôle total sur les missions et les récompenses.

Objectif : Moins de conflits, plus d’équilibre !
